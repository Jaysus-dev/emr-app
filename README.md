Mini EMR CRUD
Overview

This is a simple Electronic Medical Record (EMR) system built to demonstrate basic CRUD (Create, Read, Update, Delete) operations. It allows users to manage patient information, including admission, viewing details, editing records, and deletion.

This project is intended as a mini-assessment showcasing Laravel + Vue (or your chosen stack) skills for a basic EMR workflow.

Features

Patient Management

Add new patients (admission)

View patient details

Update patient information

Delete patient records

Simple UI

List of patients

Patient detail view

Modal-based admission form

Data Storage

MySQL database (or PhpMyAdmin-compatible)

Patient records stored in patient_lists table

Routing

Clean URL structure for listing, viewing, and editing patients

Validation

Input validation for required fields (name, age, gender, etc.)

Requirements

PHP >= 8.x

Laravel >= 10.x

Node.js >= 18.x

MySQL / MariaDB

Composer & NPM/Yarn

(Optional) Inertia.js + Vue 3 for front-end interactivity
